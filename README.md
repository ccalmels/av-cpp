# av-cpp
Simple C++ API for ffmpeg libraries

## Build and Installation

Dependencies:

```console
# apt install libavformat-dev libavcodec-dev libswresample-dev libavutil-dev
```

Meson build system is used:

```console
$ meson build
$ ninja -C build test
```

Check the test/example programs for usage.
